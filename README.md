📝 Task Management Web Application

A responsive task management web app built using HTML, CSS, and JavaScript. It allows users to create, update, and manage daily tasks with a simple and interactive interface.

🚀 Live Demo

👉 https://your-username.github.io/todo-app-js/

👉 https://github.com/chlaxminarasimha/todo-app-js

🚀 Features
✅ Add, edit, and delete tasks (CRUD operations)
💾 Persistent storage using Local Storage
🔄 Dynamic UI updates without page reload
📱 Fully responsive design
✔️ Mark tasks as completed
